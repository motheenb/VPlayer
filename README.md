# VPlayer 2.0
Speech to text play & control YouTube audio! (Python)
**Commands:**
`-> play (song name)  # play audio from youtube
-> add (song_name)  # add song to queue
-> next song in queue if exists
-> stop (will stop song)
-> volume (option)
--> mute
--> pause
--> resume
--> low, medium, high
--> customer # input`
